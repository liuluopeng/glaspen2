玻璃数位笔:

![](./icon.svg)

实现了键鼠和数位板的分离.  
相当于在显示器前加了一面玻璃做笔记.

![](./demo.gif)

普通的数位笔在平时会被操作系统当成鼠标, 虽然比较方便, 但是本人不希望数位笔有鼠标的作用.  
适用于着急不想等待笔记软件打开 简单记录几个字 临时涂鸦 等场景.

平台:

- Mac OS

使用:

- 根据引导,在设置中添加辅助功能和录屏权限.
- 在顶部的工具栏可选择画笔颜色, 画笔粗细.
- 清屏快捷键是: `Cammand + Control + C`
- 可导出成xournal格式
- 截屏保存路径是: `桌面`, 截图可选携带背景和不携带背景.
- 可以在数位板厂家的驱动软件设置中绑定一个快捷键, 快速打开APP.

注意:

- 玻璃数位笔开启时, xournal++等笔记APP的数位笔失效.
